멤버십 가입 방법

리서치알음 멤버십가입, 추천인 입력 방법

  1. 회원가입

    로그인 창 우측 하단
    "멤버십가입 방법"
    버튼을 눌러주세요

  2. 약관동의

    이용약관에 동의함을
    클릭해 주세요

  3. 본인인증

    본인명의의 핸드폰으로
    본인인증을 진행해 주세요

  4. 인증번호 기입

    정보 기입 후
    인증번호 입력하면 완료

  5. 회원정보 입력

    아이디, 비밀번호, 닉네임 등
    회원정보를 입력해 주세요

  6. 추천인 선택하기

    추천인이 있다면
    "추천인 아이디 / 닉네임 / 성함 /
    휴대폰번호"     입력해 주면 완료

  7. 멤버십 결제방법
    선택하기

    추천인 정보 입력 후
    결제수단 선택
    가입하기버튼을 클릭하면
    알음 멤버십 가입 완료

    텔레그램 초대는 매일 아침 9시에
    일괄전송됩니다.

    매주 발간되는 리포트 무제한
    열람 가능
